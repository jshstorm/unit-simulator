- 코어 포팅 비용 < 프로덕션 운영 비용 차이

- 리소스 수량 서버 관리
- 위변조 방지

## Feature #4: 전투 — 클래시 로얄 방식 RTS 타워 공방

**우선순위**: 높음 (기존 시뮬레이션 코어의 직접 확장)

기존 시뮬레이션 코어를 확장하여 클래시 로얄 스타일 전투 구현.

### 핵심 요소
- 영웅 유닛 직접 제어 (고유 스킬, 레벨링)
- 소환수 생성 타워 (주기적 유닛 스폰)
- 타워 공방 (공격/방어 타워)
- 마나/엘릭서 자원 관리 (시간 경과 증가 → 배치 비용)
- 카드 덱 시스템 또는 유닛 배치 슬롯 시스템

### 기존 코어에서 재사용 가능
- 유닛 이동/전투, A* 경로찾기
- 웨이브 스폰, 커맨드 큐
- 타워 시스템, 어빌리티 시스템
- 결정론적 시뮬레이션 프레임워크

### 추가 구현 필요
- 영웅 유닛 시스템 (직접 제어, 고유 스킬셋)
- 실시간 자원 관리 (엘릭서/마나)
- 카드/덱 메커닉
- 타워 배치 영역 시스템
